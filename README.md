conpinion-circular-image-button
===============================

This component displays a rounded image button.


Properties
----------

  * **src**
  
    *type*: String
    *default*: undefined
  
    The url of the image
  
  * **mini**
  
    *type*: Boolean
    *default*: false
  
    If true, a smaller version of the button is displayed
  
  * **elevation**
  
    *type*: Number
    *default*: 1
      
    WHen greater than zero, a shadow is displayed
